# Goodminton 学员反馈模板

这个模板服务于学生页，不是单纯的档案表。每个字段都要能回到反馈系统：课堂观察、学生感受、家庭作业、比赛复盘、能力矩阵、技能树、成就。

## 建档方式

1. 复制 `data/student-template.json`。
2. 改名为 `data/students/{studentId}.json`。
3. `studentId` 使用拼音，例如 `xue-meijiao`。
4. `accessCode` 单独设置，不和学生姓名一起公开。
5. 真实学员 JSON 不提交到 GitHub，更新后同步到 Vercel 环境变量。

## 页面字段

`level`

内部等级可继续写爱好者评级，例如 `V`、`A1`、`A2`、`B1`、`B2`、`C1`、`C2`。学生页会换算成游戏化段位：

- 青铜
- 白银
- 黄金
- 铂金
- 钻石
- 星耀
- 最强王者
- 荣耀王者

当前默认映射。先看内部等级，只有没有明确等级时才用 `progress` 兜底：

- `V`：荣耀王者
- `A1`：最强王者
- `A2`：星耀
- `B1`：钻石
- `B2`：铂金
- `C1`：黄金
- `C2` / `D`：白银
- 更基础或未评级：青铜

`progress`

总进度，0-100。它会影响顶部进度显示，也作为等级换算的辅助依据。

`abilityMatrix`

固定六项：

- 身体
- 心理
- 态度
- 技术
- 战术
- 步法

每项 1-8 档，8 档最高。这个是教练主观评估加课堂证据，不追求一次精确，后面每几节课校准。

`skillTree`

固定大类：

- 发接发
- 前场
- 中场
- 正手后场
- 头顶后场
- 反手后场
- 步法
- 心理
- 身体
- 战术

每个节点只用三种状态：

- `done`：已经稳定掌握
- `active`：当前训练中
- `locked`：还没开始或不稳定

柱形图高度也是 1-8 档，8 档最高。高度由 `done + active` 的数量决定。比如“前场第 5 档”，就是前 4 个节点 `done`，第 5 个节点 `active`，后面 `locked`。

`lessonSummary`

当前课后反馈区。学生自己填写 `studentReflection`，教练填写 `coachNote` 和 3 条家庭作业。家庭作业只保留这一处，避免页面重复。

`lessonHistory`

上课日志，按日志方式滚动显示。每节课至少要有：

- date
- title
- mainContent
- coachNote
- studentNote
- homeworkDone
- homeworkTotal

`matchReview`

比赛复盘由学生自己填。默认可以留空，页面提供输入框：

- 比赛 / 对手
- 比分
- 有效的地方
- 待改进的点
- 积累的经验

`achievements`

成就和勋章要来自真实行为，不做装饰。数量可以多，学生页会滚动显示。默认覆盖这些方向：

- 学习笔记：能把课堂重点用自己的话记录下来
- 训练态度：愿意试错、愿意复盘
- 稳定出勤：保持稳定上课节奏
- 比赛狂：愿意从实战中发现问题
- 练球达人：持续完成练习任务
- 课堂专注：抓住本节课最重要的一个点
- 主动提问：把不明白的地方说出来
- 反馈闭环：课堂、作业、复盘连成循环
- 不找借口：失误后先找可改动作
- 听得进改得动：把教练反馈转成动作变化
- 连续到课：保持训练节奏不断档
- 发接发猎手：发接发阶段有主动选择
- 网前压迫：前场开始制造压力
- 中场护盾：平抽、挡杀和接杀稳定
- 正手后场炮台：正手后场具备进攻和变化
- 头顶突击：头顶区能完成移动后的击球选择
- 反手脱困：反手后场具备脱困能力
- 步法引擎：启动、移动、衔接、回动逐步连成系统
- 速度开关：能切换身体节奏
- 心理锚点：比分波动时能回到清晰动作重点
- 线路阅读者：能读出对手线路和下一拍意图
- 三局记忆：三局赛后能留下经验和调整点
- 首条复盘：完成第一次课后总结
- 复盘连续性：连续留下课后反馈
- 作业连续完成：连续完成家庭作业
- 课前准备：课前完成自检和准备
- 技术采集者：技能树产生稳定掌握点
- 网前手感：前场处理方式变多
- 后场发动机：后场发力链条建立
- 步法底盘：启动、移动、回动、调整形成连接
- 身体火花：身体素质进入训练状态
- 平分冷静：比赛压力下能做清晰选择
- 战术镜头：能从回合里看出选择
- 比赛记忆：沉淀一条可复用比赛经验

## 调整数值的最短格式

可以直接这样给：

```text
美姣：技术 65，步法 68，身体 75，心理 85，比赛 72，战术 50；
发接发第3档，前场第5档，正手后场第4档。
```

然后只改对应 JSON，并同步线上环境变量。
